Programa de quinielas en Excel

Hoja "Equipos"
En esta hoja se deben introducir los encuentros de cada semana para que aparezcan el página web de captura.
Hoja "Quinielas"
En esta hoja se insertan los datos de los participantes mediante la captura en la página web
Hoja de Resultados
Esta hoja es la que contiene todas las fórmulas para saber la puntación de todos los participantes.

Páginas HTML del programa

Página de captura.
Esta página web es la encargada de leer la información de la hoja "Equipos" y nos permite capturar el nombre del participante, así como la quiniela de cada participante. Finalmente incrusta toda la información en la hoja denominada "Quinielas".
Página que realiza el proceso de datos y la creación del archivo PDF.
Esta página web nos permite realizar el proceso de cálculo de todas las quinielas de todos los participantes al igual que la generación del archivo en formato de PDF.

Órden cronólogico del programa

1. Captura de Datos

El administrador del programa captura o edita la hoja denominda "Equipos" con los encuentros de la jornada, ya que la página web de captura se basa en esta información.

2. Captura de datos de la quiniela.

Los participantes capturan su nombre y hacen sus elecciones de los partidos de la jornada. Esta página envía la información a la hoja denominada "Quinielas".

3. Hoja "Quinielas".

Una vez que se envían los resultados, estos se muestran en la hoja "Quinielas".

4. Cálculo del proceso y envío de información.

En este paso se lleva a cabo el proceso de cálculos y se genera el envío de los resultados en formato PDF.
