Programa de Quinielas

  • Plantilla de Partidos

    Esta plantilla de Excel contiene los encuentros de la jornada. Cada semana se debe de editar para que la página web los lea.

  • Página web de captura.

    Esta página web lee los partidos de la hoja "Equipos", nos permite la captura de la información y guarda los datos en la hoja "Quinielas".

  • Base de datos de quinielas

    Esta hoja es la que contiene la base de datos que se crea con la captura de todos los participantes.

  • Página web de procesos.

    Esta página web realiza los procesos de todos los cálculos y crea un archivo con toda la información final y lo envia a un correo.

  • Hoja de fórmulas y procesos.

    La hoja de "Resultados" es la que contiene todas las fórmulas, con estos datos se crea el archivo en formato PDF.
